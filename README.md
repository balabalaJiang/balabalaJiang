# 👋 你好，我是江永閎

### 🎓 關於我

目前就讀於 **中山醫學大學 (CSMU) 醫學資訊學系**。我致力於結合醫療領域知識與人工智慧技術，專注於開發輔助診斷系統與大數據分析流程。

* 🔭 **專業領域**：醫療影像 AI、深度學習、大數據工程。
* 🌱 **當前實踐**：優化卷積神經網絡 (CNN) 於臨床影像的表現，並利用 SQL 進行大規模數據預處理。
* ⚡ **技術特點**：擅長整合不同技術棧，從深度學習模型開發到基於 Node-RED 的自動化監控儀表板。

### 📂 精選技術專案

#### 🩺 1. [胸部 X 光影像分類 (Chest X-Ray Classification)](https://github.com/balabalaJiang/Chest-XRay-Classification-DL)
> 利用深度學習針對胸部 X 光片進行多類別病灶偵測。
* **技術棧**：Python, PyTorch/TensorFlow, Computer Vision
* **重點**：處理醫療數據不平衡問題，提升模型在肺炎辨識上的精準度。

#### 📊 2. [大數據 SQL 機器學習專案](https://github.com/balabalaJiang/BigData-SQL-ML-Project)
> 整合資料庫管理與預測建模，實現大規模數據的特徵工程。
* **技術棧**：SQL, Machine Learning, Data Engineering
* **重點**：設計高效 ETL 流程，將海量原始數據轉化為具預測價值的特徵集。

#### 🚀 3. [太空 AI 儀表板 (Space-AI-Dashboard-NodeRED)](https://github.com/balabalaJiang/Space-AI-Dashboard-NodeRED)
> 使用 Node-RED 打造的即時數據流監控系統，展現自動化整合實力。
* **技術棧**：Node-RED, JavaScript, Dashboard Integration
* **重點**：實現資料視覺化與自動化告警流程，優化系統監控體驗。
