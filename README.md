### 🎓 關於我

目前就讀 **中山醫學大學 (CSMU) 醫學資訊學系**。我致力於結合 **醫療領域知識** 與 **人工智慧技術**，解決醫學影像診斷與數據處理的挑戰。

* 🔭 **專業領域**：醫療影像 AI、深度學習、大數據分析。
* 🌱 **技術實踐**：開發輔助診斷系統，並整合 DICOM、ICD-10 等醫療標準。
* ⚡ **自動化愛好**：運用 Low-code 工具打造即時監控儀表板與生活自動化腳本。

### 📂 精選開發專案

#### 🏆 1. [胸部 X 光影像分類 (Chest X-Ray Classification)](https://github.com/balabalaJiang/Chest-XRay-Classification-DL)
> 運用深度學習模型 (CNN) 針對胸部 X 光片進行病灶偵測與多類別分類。
* **技術棧**：Python, PyTorch/TensorFlow, Computer Vision
* **實作細節**：處理醫療數據類別不平衡問題，優化模型在肺炎等病灶上的辨識精準度。

#### 📊 2. [大數據 SQL 機器學習專案](https://github.com/balabalaJiang/BigData-SQL-ML-Project)
> 整合 SQL 資料庫管理與機器學習流程，實現大規模數據的特徵工程與預測建模。
* **技術棧**：SQL, Machine Learning, Data Engineering
* **實作細節**：撰寫高效 SQL 腳本進行大規模數據清洗，並建構模型分析海量資料。

#### 🚀 3. [太空 AI 儀表板 (Space-AI-Dashboard-NodeRED)](https://github.com/balabalaJiang/Space-AI-Dashboard-NodeRED)
> 利用 Node-RED 打造的高效自動化儀表板，整合太空 AI 應用與即時數據流監控。
* **技術棧**：Node-RED, JavaScript, IoT Integration, Dashboard
* **實作細節**：實現資料視覺化與自動化通知流程，展現 Low-code 開發的靈活性。
